Olá tudo bem? Neste mini-projeto, eu e um colega de classe desenvolvemos um algoritmo em Java, utilizando o básico de POO e bibliotecas data e tempo.
O objetivo desse programa foi aprender a utilizar bibliotecas de data e tempo, juntamente com inputs.
O algoritimo foi divido em 3 seções principais: 
(1) Exibir a data e horário em tempo real. 
(2) Receber um input em segundos. 
(3) Calcular e adicionar o input no  de acordo tempo atual.
